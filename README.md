# andrejusk/dotfiles

Collection of experimental dotfiles and supporting install scripts.

## Install

    wget https://raw.githubusercontent.com/andrejusk/dotfiles/master/scripts/setup.sh -qO - | bash

## Stack

### Shell
🐟 fish (+ fisher)

### Editor
neovim (+ vim-plug)

### Languages

- js (nvm, node, yarn)
- python (pyenv, poetry)
- java

### Tools

- docker (+ docker-compose)
- terraform
- gcloud, firebase, awscli
- screenfetch
